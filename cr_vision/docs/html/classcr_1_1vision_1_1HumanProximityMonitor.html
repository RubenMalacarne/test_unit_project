void cr::vision::HumanProximityMonitor::human_poses_callback ( const geometry_msgs::msg::PoseArray::SharedPtr  msg)
private

Callback function that processes an array of human poses.

Computes the distance between the robot's end-effector (tool0) and each human pose to determine if a human is within the configured safe distance. Publishes the result as a boolean on the /cr/human_near topic.

Parameters
msgShared pointer to the received PoseArray message.
